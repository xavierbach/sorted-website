Features

Everything you need to stop missing dates

Finds dates fast

Sorted reads newsletters, PDFs, photos, screenshots and web pages, then pulls out dates even when they are written in plain language like "the last Friday of term".

Built with modern AI

Private by design

No account required. No tracking. Imported content is processed to extract events, then discarded.

Every format supported

Photos, PDFs, screenshots, emails and web pages. However it comes in, Sorted can work with it.

Profiles for every child

Set up a profile for each child so family dates stay organised and easier to scan at a glance.

Reminders and quick capture

Add reminders as you save, and use the widget when you need to grab a date quickly on the fly.

Emoji event titles

Helpful emoji make your calendar quicker to scan, so school and family events stand out straight away.
